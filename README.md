# SmmLoLWebsite
Website for Samarnmitr League of Legends tournament.

## Features
- Tournament registration
- Bracket
- Announcement
- Static pages for info & rules

## Requirements
- Apache 2 with mod_rewrite enabled
- PHP 7 or newer
- MySQL 5.6 or newer
